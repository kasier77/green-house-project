# green-house-project
its a project for a automatic green house  made by ABDULRAHMAN MASOOD ,ZEYNEP PANAHİ AND DJONKOU RAPAHEL ADRİEN
This project is a smart irrigation and environmental monitoring system built using the PIC16F877A microcontroller.
It continuously monitors soil moisture, water tank level, temperature, humidity, and light intensity, and automatically controls a water pump while displaying real-time information on a 16×2 LCD.
The system is designed to reduce water waste and automate plant irrigation without human intervention.
